# Klyntel Website

Static website for [Klyntel](https://klyntel.io) — a company that trains domain-specific AI models and deploys them with decision integrity infrastructure.

## Pages

- **index.html** — Home
- **product.html** — Product overview
- **news.html** — News and updates
- **contact.html** — Contact
- **terms.html** — Terms of service
- **privacy.html** — Privacy policy

## Design

- **Layout**: Two-column — city SVG road map on the left, content on the right
- **Typography**: Inter (sans-serif) + Lora (serif)
- **Colors**: Slate blue palette, #fafafa background
- **City maps**: San Jose + Half Moon Bay (home), Seattle (product), Berkeley (news), San Francisco (contact), San Diego (terms), Los Angeles (privacy)
- **Responsive**: Mobile-friendly with stacked layout on smaller screens

## Structure

Static HTML/CSS files. No build process required.

## Deployment

Deployed via GitHub Pages at [klyntel.io](https://klyntel.io).
