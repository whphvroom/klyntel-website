# Klyntel Website

One-page website for Klyntel, an AI-Native Solutions Foundry.

## Overview

Clean, editorial-style website featuring a manifesto-format presentation of Klyntel's approach to building deterministic AI infrastructure for enterprise operations.

## Design

- **Layout**: Asymmetric layout with 18% left margin and technical coordinate markers
- **Typography**: Playfair Display serif font throughout for editorial feel
- **Background**: Subtle diagonal grid pattern (5% opacity) for technical aesthetic
- **Colors**: Slate blue color palette with off-white background
- **Responsive**: Mobile-friendly with breakpoints at 1024px, 768px, and 480px

## Structure

Single HTML file with inline CSS. No build process required.

## Deployment

Can be deployed to any static hosting service (GitHub Pages, Netlify, Vercel, etc.).
